# 단축키

### 모든 단축키 정보

-> ctrl + shift + l : 이클립스의 모든 단축키 안내



### 주석

* ctrl + shift + / : 블록 주석처리

* ctrl + shift + \ : 블록 주석 해제

* ctrl + / : 한줄 블록

  ​



### 소스편집

* ctrl + d : 한 줄 삭제
* alt + 방향키 : 코드 이동하기
* alt + ctrl + 방향키 : 코드 복사하기
* ctrl + shift + f : 들여쓰기 정리
* ctrl + shift + o : 내가 사용한 기능 import 
* shift + enter : 줄 중간에 커서 있는데, 다음줄로 넘어가고 싶을 때 사용 (vs code에서는 ctrl + enter)
* ctrl + y : 앞으로 되돌리기
